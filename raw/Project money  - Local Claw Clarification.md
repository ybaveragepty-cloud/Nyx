### How it actually works

There are two mechanisms:

#### 1) Automatic memory (passive)

- When memory is ON, ChatGPT decides what is worth remembering.
- It typically stores:
	- Long-term goals (e.g., “Project Money”)
		- Preferences
		- Ongoing work themes

You don’t control this directly.

#### 2) Explicit memory (active — what you want)

You tell it clearly to remember something.

- “Remember that I’m working on Project Money, which is focused on building income streams.”
