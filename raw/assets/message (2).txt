“3 users” in uptime does not mean 3 system accounts.
It means 3 active login sessions (SSH terminals, consoles, etc.).

You may still only have two actual users (root and richard), but someone could have:

multiple SSH sessions open

a background terminal

a Coolify shell session

an orphaned session

Run these commands to see exactly who is logged in.

1. See current login sessions
who
Example output:

richard  pts/0  2026-03-16 07:10 (192.168.1.10)
richard  pts/1  2026-03-16 07:15 (192.168.1.10)
root     pts/2  2026-03-16 07:20 (192.168.1.10)
This would explain the 3 users.

2. Better view of active sessions
w
This shows:

username

login time

IP address

what command they are running

Example:

USER     TTY      FROM             LOGIN@   IDLE   JCPU   PCPU WHAT

3. See login history
last
This tells you:

who logged in

when

from what IP

Good for security checks.
